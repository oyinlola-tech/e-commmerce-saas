# Customer Service

Proprietary software by Oluwayemi Oyinlola Michael. Portfolio: https://www.oyinlola.site/

This package is not free to use.

## Purpose

`@aisle/customer-service` manages storefront customer identity, authentication, and profile data on a per-store basis.

## Runtime

- Default port: `4104`
- Entry point: `server.js`
- Routes: `src/routes.js`
- Schema: `src/schema.js`

## Main Capabilities

- Store-scoped customer registration.
- Store-scoped customer login.
- Authenticated customer self-profile lookup and update.
- Store operator access to customer lists inside a store.

- Publishes `CUSTOMER_REGISTERED`.
