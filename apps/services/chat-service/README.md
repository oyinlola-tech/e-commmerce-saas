Proprietary software by Oluwayemi Oyinlola Michael. Portfolio: https://www.oyinlola.site/

This package is not free to use.

## Current Status

`@aisle/chat-service` is currently a placeholder package. The package metadata exists, but runnable service code has not yet been added in this repository snapshot.

## Intended Role

- Storefront live chat
- Owner and support messaging
- Conversation threads and message history
- Socket-based chat transport through the gateway

## Notes

- Reserved default port: `4111`
- Gateway proxy paths and socket upgrade hooks already assume this service will exist.
